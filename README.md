# QuietBar Releases

Official downloads and release notes for QuietBar, a macOS menu bar utility that helps keep crowded menu bars tidy.

## Current status

QuietBar is being prepared for its first public release. A signed download will appear here after Developer ID signing and Apple notarization are complete.

There is no public binary available yet. Do not download QuietBar from unofficial sources.

## What QuietBar does

- Expands and compacts menu bar items with one click.
- Shows detected application names and icons when Accessibility access is granted.
- Supports Chinese and English.
- Works locally on your Mac.

## Requirements

- macOS 26.5 or later.
- Accessibility permission for reading menu bar item names and positions.
